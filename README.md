# 🏢 NexusHome

A modern, cyberpunk-themed Real Estate and Property Management web application built with Next.js and Tailwind CSS.

## 🔗 Live URL
🌐 [NexusHome Live Site](https://nexushome-phi.vercel.app) 
---

## 🎯 Purpose
NexusHome is a platform that connects Property Admins, Owners, and Tenants. It allows users to easily search, book, and manage properties through dedicated, role-based dashboards.

---

## ✨ Key Features
* **Role-Based Dashboards:** Custom layouts and options for Admins, Owners, and Tenants.
* **Fully Responsive:** Beautiful cyberpunk dark theme layout that works perfectly on both Desktop and Mobile devices.
* **Property Management:** Owners can list properties, Admins can review them, and Tenants can book them smoothly.

---

## 📦 Key npm Packages Used
* **`next`** - React framework for fast development and App Routing.
* **`@heroui/react`** - Used for UI components like Avatars, Buttons, and Drawers.
* **`@gravity-ui/icons` & `react-icons`** - For clean and interactive dashboard icons.
* **Tailwind CSS** - For modern and responsive styling.
